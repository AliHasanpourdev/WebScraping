# Web Scraping Projects

مجموعه پروژه‌های عملی وب‌اسکرپینگ با پایتون برای استخراج داده‌های واقعی از وب‌سایت‌های مختلف.

## پروژه‌های داخل ریپو
- **USD/** — اسکرپینگ نرخ دلار و ارزها
- **books/** — استخراج اطلاعات کتاب
- **enamad/** — اسکرپینگ نماد اعتماد الکترونیکی
- **v3/** — نسخه پیشرفته اسکرپرها
- **enamad.ipynb** و **enamad_Scraping.ipynb** — نوت‌بوک‌های Jupyter

## تکنولوژی‌ها
- Python
- BeautifulSoup, Requests, Selenium
- Jupyter Notebook
- Pandas

## نحوه اجرا

```bash
git clone https://github.com/AliHasanpourdev/WebScraping.git
cd WebScraping

# نصب کتابخانه‌ها
pip install -r requirements.txt

jupyter notebook
```
